Category: security
SECTION: Injection
Injection flaws occur when untrusted data is sent to an interpreter as part of a command or query. Prefer parameterized APIs, allowlists, and structured command invocation.
---
SECTION: Identification and Authentication
Authentication code must protect credentials, use strong password hashing, avoid hardcoded secrets, and return tokens generated by secure services.
---
SECTION: Access Control
Object fetches based on user-supplied identifiers require authorization checks against the current user or service principal before returning sensitive data.
